# HabboHotel 2001 CMS

This package preserves the 2001-era Habbo Hotel website as a CMS generation inside Epsilon.

What is preserved:
- original HTML pages
- original image assets
- original information architecture

What is updated:
- modern wrapper entry pages for hotel launch and diagnostics
- generation config for the shared CMS system
- original entry pages moved to `site/legacy/`

Entry points:
- `site/index.html` modernized front page
- `site/loader.html` modernized client/launcher bridge
- `site/loggeduser.html` modernized online-state panel

Legacy originals:
- `site/legacy/index.original.html`
- `site/legacy/loader.original.html`
- `site/legacy/loggeduser.original.html`
